# Project Structure

- `main.py`: runs the full project.
- `load_data.py`: loads the portfolio data files.
- `cost.py`: computes return, risk, and the full cost.
- `evaluate.py`: runs trials, summaries, and plots.

# Algorithms

- `algorithms/de.py`: differential evolution.
- `algorithms/pso.py`: particle swarm optimization.
- `algorithms/sa.py`: simulated annealing.
- `algorithms/ga.py`: genetic algorithm.

# Data

- `data/`: holds the portfolio data files.
- `port1.txt` through `port5.txt`: the five OR-Library portfolio test problems already downloaded from `https://people.brunel.ac.uk/~mastjjb/jeb/orlib/portinfo.html`.
